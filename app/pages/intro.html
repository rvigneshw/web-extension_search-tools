Have ever been frustated over these small things?

We usually start our web surfing by looking for something on one of the 3 major search engine providers. These search engines provide us with a bunch of results.
But we will never just open a single link, We want to open a bunch of links because we need to get more information about the thing we're looking for! Then there comes this list of problems (or minor inconvienencies!)

  • Ability to open several tabs. We need to look at lot's of webpages, but the current method of right-clicking each time and pressing Open In New Tab is a nightmare.
  • Copying Links You can copy direct link to clipboard.Usually when you copy a link it will be like this: https://www.google.com/url?ved=2ahUKEwiI8oCNqZfqAhUPxDgGHXxTAb4QFjAAegQIBhAB&url=https%3A%2F%2Fen.wikipedia.org%2Fwiki%2FVenice&usg=AOvVaw1u5J7kugR97CpRl3MAX2Nb Which usually goes to google servers before going the required site (You guessed it ! It's called tracking your web activity). Whereas the original URL be like: https://en.wikipedia.org/wiki/VeniceUsing the above copied for sending to someone for reference or something else would obviously look terrible!
  • Save a link for future reference You may need to save a link for future reference. When you use bookmark it'll become more and more each and finding your links will be cumbersome task
  • Want to get the information about the domain before proceeding to that site? You may want to know the IP Address and which country from this particular website is being served, but usually you need to copy the link and paste it into a third party WHOIS look up provider and get the job done. Daunting isn't it?

What does this extension do?

This extension provides you with neccessary shortcut actions which appear below every search result, enabling you to get rig of all the above mentioned frustations!

  • Open Links in New Tab with just single click.
  • Copy links (Without any additional tracking stuffs) to clipboard with single click
  • Open the link securely without google/bing/duckduckgo tracking you!
    • Add a particular link to collection, those links will be automatically categorised under the search keyword so you don't have to go through bunch of URL's to find a particular one. With a single click.
  • Get the WHOIS information about a domain like IP Address, Country name etc... With a single click.
  • And a special one??
  • You can directly go to the answers section of stackoverflow page (you no more need to scroll thorught long and lengthy question each time, Hurray!)
